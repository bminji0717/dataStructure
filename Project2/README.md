# GCD, LCM ADT

## GCD
### 최대공약수
### 유클리드 호제법 이용
#### 유클리드 호제법: 두 정수 x, y 중 큰 수를 작은 수로 나눈다.
####	그 나머지가 0이 아니라면, 나누었던 수를 나머지로 나눈다.
####	나머지가 0이 될 때까지 반복.
####	나머지가 0이 되는 수가 두 수의 최대공약수.
### while 구문 이용
#### 두 수 x, y를 받아 if 구문으로 큰 수를 구분 후 계산
